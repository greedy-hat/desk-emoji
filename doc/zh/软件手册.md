## 安装

* 从 [Python官网](https://www.python.org/) 下载python并安装：

  * Python 版本推荐在3.9～3.11之间
* 首先初始化虚拟环境：

  * 对于 MacOS 或 Linux:

    * 打开终端，执行命令： `./init.sh`
    * 激活虚拟环境： `source .venv/bin/activate`
* 对于 Windows:

  * 打开终端，执行命令： `.\init.bat`
  * `激活虚拟环境：.venv\Scripts\activate`
* 启动主程序

  * `python main.py`
* 测试代码

  * Test action: `python action.py`
  * Test chat:  `python chat.py`
