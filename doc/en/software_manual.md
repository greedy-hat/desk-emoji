## Installation

* 3.9 <= Python version <= 3.11
* Initialize environment

  * For MacOS or Linux:

    * `./init.sh`
    * `source .venv/bin/activate`
  * For Windows:

    * `.\init.bat`
    * `.venv\Scripts\activate`
* Launch the main program

  * `python main.py`
* Test code

  * Test action: `python action.py`
  * Test chat:  `python chat.py`
