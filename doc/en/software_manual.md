## Installation

* Download and install Python from [python.org](https://www.python.org/):

  * 3.9 <= python version <= 3.11

* Initialize environment

  * For MacOS or Linux:
    * `./init.sh`
    * `source .venv/bin/activate`

  * For Windows:
    * `.\init.bat`
    * `.venv\Scripts\activate`

* Launch the main program
  * `python main.py`
