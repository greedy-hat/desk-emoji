# Desk-Emoji

## 概述

* 首款工业风桌面机器人，外形炫酷，可作桌搭摆件。
* 极致性价比，百元实现千元级桌面机器人效果。
* 具备2自由度云台，丰富的头部运动，是一款真正可动的桌面机器人。
* 精心调校的 Emoji 表情动画和运动算法，流畅而生动，情绪价值拉满。
* 可根据回复内容的情绪做出相应的动作反馈。
* 支持手势识别互动。
* 支持大模型语音对话。
