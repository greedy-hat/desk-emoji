# Desk-Emoji

## 概述

Desk-Emoji 是一款开源的实体 AI 桌面陪伴机器人，丝滑动画表情、2自由度云台、LLM语音聊天功能。 ( [演示](https://www.bilibili.com/video/BV1GnsdePEaz/) )
